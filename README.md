#  BelíssimaApp

Aplicativo Android desenvolvido em **Kotlin** para controle de produtos da Belíssima Cosméticos.

## 📱 Sobre o Projeto

O BelíssimaApp é um aplicativo simples para gerenciamento de produtos, permitindo visualizar e organizar itens de forma prática e intuitiva.

Este projeto foi desenvolvido com foco em:

- 📦 Organização de produtos
- 🎨 Interface moderna com Jetpack Compose
- 📋 Listagem dinâmica de itens
- 🧩 Estrutura limpa e organizada

---

## 🚀 Tecnologias Utilizadas

- Kotlin
- Android Studio
- Jetpack Compose
- Material Design 3
- Gradle (KTS)
